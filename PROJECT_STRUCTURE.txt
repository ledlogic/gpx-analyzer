Package: com.github.ledlogic.gpxanalyzer

Main Classes:
  - TrackPoint: Represents a GPS point with coordinates, altitude, and distance
  - GPXElevationProfile: Parses GPX files and calculates distances
  - ElevationPlotter: Creates graphical elevation profile charts
  - ElevationProfileApp: Main entry point with command-line interface

Quick Start:
  1. ./compile.sh
  2. ./run.sh sample_track.gpx

Full Command:
  java -cp bin com.github.ledlogic.gpxanalyzer.ElevationProfileApp <gpx-file> [options]
